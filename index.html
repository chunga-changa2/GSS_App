GSS App (Beta test) v2.0

Приложение для электронного заполнения сопроводительных документов (бирок) для проб угля и не только.

Современно, актуально, удобно, аккуратно технологично.

Приложение рассчитано на работу только на ПК или ноутбуке.

Инструкция к применению.

  • Заполнение всех полей вручную.
  • Заполнение путем Пресета, частичное заполнение путем заранее заготовленного варианта.
    • Примечание к пункту
  • Выбор конкретной бирки, которую необходимо заполнить (распечатать)
  • Итоговые данные для внесениея в журнал входящих проб

  • Кнопка распечатать
  • После распечатки
  • Сейчас в проекте

Заполнение всех полей вручную

Поля заполняются вручную сверху в низ, Обязательных полей нет. (в планах сделать все поля обязательными) смотрите пункт Что планируется внедрить.

  • В поле Номер партии допускается ввод только цифры и не более 4-х символов.
  • В поле Дата можно заполнить вручную в формате (01.02.2025) или нажать на дату в выпадающим календаре. По умолчанию значение пусто.
  • В поле Место отбора проб заполняется путем выбора из выпадающего списка. По умолчанию значение пусто.
  • В поле Номер накладной заполняется вручную, первые два символа должны быть кириллические символы Русского алфавита, Caps Lock применяется автоматически, остальные 6-ть символов должны быть цифры. Если в партии более чем одна накладная, то ниже есть кнопка, которая добавит поле для следующей накладной и так до 6-и накладных.
  • В поля Серийный номер пломбы (4-е поля.) заполняется вручную согласно их реальным серийным номерам, первые два символа латинские буквы английского алфавита, Caps Lock применяется автоматически, остальные 6-ть символов должны быть цифры.
  • В поле Метод отбора пробы заполняется путем выбора из выпадающего списка. По умолчанию выбрано значение ISO.
  • В поле Марка и класс крупности заполняется путем выбора их выпадающего списка. Значение по умолчанию пусто.
  • В поле Клиент заполняется путем выбора из выпадающего списка. Значение по умолчанию пусто.
  • В поле Масса брутто лабораторной пробы (4-е поля.) заполняется вручную, допускается ввод только цифр. Не более 4-х символов.
  • В поле Количество вагонов заполняется вручную, допускается ввод только цифр. Не более 2-х символов.
  • В поле Вес нетто партии заполняется вручную согласно реальной ЖД накладной текущей партии. Допускается ввод только цифр.
  • В поле Фамилия ответственного лица заполняется путем выбора из выпадающего списка. Значение по умолчанию пусто.

Заполнение путем Пресета
частичное заполнение путем заранее заготовленного варианта

Пресет это заранее настроенная комбинация
Выбранный пресет избавляет от необходимости заполнять все поля формы так как он автоматически заполняет 4 поля из формы и делает их не активными, остается лишь заполнить все активные поля дописав их в приложении или распечатав вручную шариковой ручкой.

Как им воспользоваться:

  • В верхней левой части приложения нажмите на шестерёнку с надписью настройки, из выпадающего списка в пункте пресеты выберете из выпадающего списка нужный пресет.

Выбор конкретной бирки, которую необходимо заполнить (распечатать)

Из 5-ти бирок можно выбрать конкретную позицию, которую можно вывести на печать. Например, есть необходимость распечатать только две бирки Расчетная – 2 и Контрольная

Как это сделать:

  • В верхней левой части приложения нажмите на шестеренку с надписью настройки и в выпадающем списке в пункте (Выберете документ, который хотите распечатать) снимите галочки с тех пунктов, которые не нужно выводить на печать. По умолчанию все пункты включены.

Итоговые данные для внесениея в журнал входящих проб

В данный раздел выводятся все данные, внесенные пользователем в сокращенном виде для внесения в журнал входящих проб.

После распечатки

После того как документ распечатан в полях, где стоят галочки нужно расписаться исполнителю и из итоговых данных заполнить журнал входящих проб.

Что планируется внедрить

Адаптация под мобильные устройства для работы с телефона или планшета

  • Для реализации не обходимо будет оснастить рабочие места мобильным принт-сервером (печать с мобильных устройств)

Сделать все поля обязательными дабы свести к минимуму ошибку исполнителя. Когда заполняют вручную шариковой ручкой бывает, что исполнитель часто забывает написать, например вес партии или дату или номер партии.

Переделать форму для заполнения, текущая форма — это так для понимания как это работает и без какого-либо дизайна.
